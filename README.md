# credit-score-classification
Diseñé un pipeline de preprocesamiento reproducible que normaliza las variables financieras con técnicas robustas a outliers y codifica variables categóricas, y guardé el scaler entrenado para poder aplicar exactamente las mismas transformaciones en producción.
